# Yachiyo
本仓库为个人项目仓库(A desktop AI companion inspired by Yachiyo.)
本人在观看完《超时空辉夜姬》后感叹百合真香啊！()
于是有了想把八千代邀请到自己的桌面的这个想法，感觉蛮有意思的，遂尝试。

##关于项目
因为这是本人尝试做的第一个项目，所以会在AI的指导下尝试完成。
项目期望实现的功能：我想做一个可以在桌面展示的那种小皮套人（角色是八千代，或者其他角色？），可以有表情的变化（类似于主播直播时的那种皮套人）。然后可以进行一些简单的交互（比如我可以摸她头她就会有开心的那种表情，其他的暂时没想到）。至于一些在桌面移动的动作模组似乎目前还只能畅想一下，先把基础版做出来。我与这个皮套进行进行交互时她回说一些日文（声音要和八千代相差无几，这里是否涉及到一些模型训练得到相应语音，似乎普通的tts效果一般，语音比较死板，之前做项目感觉到过，没有那种活人味）。最后我在想要不要给她接一个聊天窗口然后接上大模型，可以跟她进行交流，她同样可以进行回答。
